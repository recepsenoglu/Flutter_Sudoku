# Flutter Sudoku Game App

A Sudoku game app built with Flutter. The app is still under development.

## Getting Started

### Dependencies

- Flutter
- Dart

## Installation

Clone the repository and run the following command:

```bash
flutter pub get
```

## Usage

Run the following command to run the app:

```bash
flutter run
```
